## Selected domain: **Makeup**
Makeup is a constantly evolving and highly trend-driven field, with new products, techniques, and styles emerging every day. I chose this domain because it offers a wide variety of themes to explore, ranging from everyday looks to high-fashion artistry. Its dynamic nature allows for countless possibilities when it comes to queries, discussions, and creative expression.

---

## How to run project

### 1. Necessary installations
```bash
pip install -r requirements.txt
```

### 2. Initiate server
```bash
python app.py
```

### 3. Open in web search
```
http://localhost:5000
```

## Enhancement
C - Autocomplete
Term suggestions as the user types in the search bar
For autocomplete, words are taken from the inverted index. Among all the keys stored in the inverted index, those that start with the first letters typed by the user in the search bar are suggested (first five).
